# Анализ данных пассажиров «Титаника» с использованием Pandas и Matplotlib

**Ссылка на датасет:** [Titanic Dataset](https://www.kaggle.com/c/titanic/data)

## Этап 1. Загрузка и предварительный обзор данных

1. Импортируйте данные из файлов `train.csv` и `test.csv` и объедините их в единый `DataFrame`.
2. Проведите первичный анализ данных:

   * Используйте `.info()` для изучения структуры и типов данных.
   * Используйте `.describe()` для получения базовой статистики.
   * Просмотрите первые строки с помощью `.head()`.
3. Составьте краткое текстовое описание каждого признака и его предполагаемой роли в анализе.

## Этап 2. Очистка и предобработка данных

1. **Обработка пропусков:**

   * Рассчитайте процент пропусков по каждому признаку.
   * Для поля `Age` выполните импутацию медианным значением для каждой комбинации `Pclass` и `Sex`.
   * Для поля `Embarked` выполните импутацию модой.
   * Исключите столбец `Cabin` из дальнейшего анализа из-за большого числа пропусков.
2. **Обработка дубликатов:**

   * Проверьте датасет на полностью повторяющиеся строки и удалите их при необходимости.
3. **Анализ выбросов:**

   * Выявите выбросы в поле `Fare` с помощью метода IQR и отметьте их наличие.
4. **Создание производных признаков:**

   * Категориальный признак `AgeGroup`: `Child` (<12), `Adult` (12–60), `Elderly` (>60).
   * Числовой признак `FamilySize` = `SibSp` + `Parch` + 1.
   * Бинарный признак `IsAlone` (`True`, если `FamilySize == 1`).

## Этап 3. Разведочный анализ и проверка гипотез

Для проверки гипотез создайте визуализации с помощью Matplotlib и сопроводите их кратким выводом.

## Этап 4. Создание итогового отчета

1. Объедините ключевые визуализации из Этапа 3 в единую фигуру с помощью `plt.subplots()`.
2. Расположите графики в логической последовательности, соответствующей проверяемым гипотезам.
3. Добавьте общее название отчета и подписи к каждому графику.
