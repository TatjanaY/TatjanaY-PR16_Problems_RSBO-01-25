# Анализ эффективности рекламных кампаний в digital-маркетинге

**Ссылка на датасет:**

* **Online Marketing Campaign Dataset** — данные о многоканальных кампаниях, кликах, показах, расходах и конверсиях:
  👉 [https://www.kaggle.com/datasets/saurabhshahane/online-marketing-campaign](https://www.kaggle.com/datasets/saurabhshahane/online-marketing-campaign)

**Примечание:** Можно использовать данные из CSV, которые содержат основные показатели кампаний по каналам (Facebook, Google, Instagram) и пути пользователей.

---

## Этап 1. Загрузка и предварительный обзор данных

* Загрузите CSV-файл в Pandas DataFrame.
* Проверьте структуру данных:

  * `.info()` — типы и структура признаков
  * `.describe()` — базовая статистика числовых полей
  * `.head()` — первые строки
* Составьте краткое описание каждого признака:

  * Кампания, Канал, Показы, Клики, Расходы, Конверсии, User_ID, последовательность касаний.
* Определите ключевые метрики для анализа: CTR, CPA, Conversion Rate, ROAS.

---

## Этап 2. Очистка, предобработка и обогащение данных

* Преобразуйте даты в формат `datetime` и убедитесь, что нет пропусков.
* Рассчитайте производные показатели:

  * `CTR` = Клики / Показы
  * `CPA` = Расходы / Конверсии
  * `ConversionRate` = Конверсии / Клики
  * `ROAS` = Доход / Расходы (если доступен доход)
* Проанализируйте распределение числовых признаков и категориальных полей (`Канал`, `Кампания`) на наличие выбросов и редких категорий.

---

## Этап 3. Разведочный анализ и проверка гипотез

| № | Гипотеза                                                                 | Визуализация                                     | Тип графика                                     |
| - | ------------------------------------------------------------------------ | ------------------------------------------------ | ----------------------------------------------- |
| 1 | Эффективность канала (CPA) имеет недельную сезонность                    | Средний CPA по дням недели для каждого канала    | Линейный график (Seaborn lineplot)              |
| 2 | Модель атрибуции перераспределяет вклад каналов                          | Сравнение Last-Click, Linear, Time Decay         | Сгруппированная столбчатая диаграмма            |
| 3 | Анализ путей пользователей выделяет типичные паттерны поведения          | Кластеризация последовательностей пользователей  | Кластерная визуализация / Bar Plot по кластерам |
| 4 | Существуют устойчивые комбинации каналов, приводящие к высокой конверсии | Частотные комбинации каналов в пути пользователя | Столбчатая диаграмма / Heatmap                  |
| 5 | Метрики (CTR, Conversion Rate) зависят от времени суток и дня недели     | Heatmap CTR по часам дня и дням недели           | Тепловая карта (Seaborn heatmap)                |

* К каждому графику добавьте краткий вывод о подтверждении или опровержении гипотезы.

---

## Этап 4. Создание итогового аналитического отчета

* Объедините ключевые визуализации из Этапа 3 в единый отчет.
* Расположите графики в логической последовательности по проверяемым гипотезам.
* Добавьте:

  * общее название отчета;
  * подписи осей;
  * поясняющие заголовки к каждому графику.
* Дополнительно: можно создать интерактивную панель с `ipywidgets` для выбора канала, метрики и временного периода с динамическим обновлением графиков.

---

## Этап 5. Заключение

* Подведите итоги по каждой проверенной гипотезе (подтверждена / опровергнута / частично подтверждена).
* Сформулируйте общие выводы о паттернах эффективности digital-кампаний и поведении пользователей.
* Дайте практические рекомендации маркетинговой команде:

  * распределение бюджета между каналами с учетом сезонности;
  * интерпретация данных атрибуции;
  * оптимизация путей пользователей для повышения конверсии.
