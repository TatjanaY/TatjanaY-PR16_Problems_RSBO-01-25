# Анализ рынка труда и тенденций трудоустройства

**Ссылка на датасет:** [Job Dataset (8K+ Job Descriptions)](https://www.kaggle.com/datasets/mohamedharris/job-dataset)

## Этап 1. Загрузка и предварительный обзор данных

* Загрузите CSV-файлы с данными о вакансиях (`job_descriptions.csv`, `job_categories.csv`, `skills_data.csv`) в Pandas DataFrame.
* Проверьте структуру данных:

  * `.info()` – типы и структура данных.
  * `.describe()` – базовая статистика числовых признаков.
  * `.head()` – первые строки для визуального осмотра.
* Краткое описание столбцов:

  * Вакансии: `Title`, `Company`, `Location`, `Description`, `Requirements`, `Salary_Range`, `Experience_Required`.
  * Категории вакансий: `Category_Name`, `Parent_Category`.
  * Навыки: `Skill_Name`, `Category`, `Demand_Trend`.
* Определите основные показатели для анализа: средняя зарплата, требуемый опыт, популярные навыки.

## Этап 2. Очистка и предобработка данных

* Проверка пропусков и дубликатов:

  * Обработайте пропуски в числовых полях (зарплата, опыт) и категориальных данных (локация, категория).
  * Удалите полностью повторяющиеся строки.
* Обработка текстовых полей:

  * Очистка описаний и требований от HTML-тегов, специальных символов, приведение регистра к нижнему.
* Создание производных признаков:

  * `min_experience`, `max_experience` — извлечение числовых значений из поля опыта.
  * `avg_salary` — средняя зарплата для вакансии с диапазоном.
  * Бинарные признаки ключевых навыков (`Python`, `SQL`, `Data Analysis` и т.д.).
* Агрегация по регионам, отраслям и временным промежуткам для дальнейшего анализа.

## Этап 4. Анализ профессий по навыкам (ранее кластеризация, без сложных моделей)

* Подготовка данных:

  * Матрица "вакансия-навык" с бинарными значениями (1 — упоминается, 0 — нет).
  * Выберите топ-50 наиболее востребованных навыков.
* Анализ:

  * Определите наиболее характерные навыки для групп вакансий (например, Data Science, Web Development, DevOps, Business Analytics).
  * Визуализируйте распределение навыков по отраслям и зарплатам.
* Выводы:

  * Какие навыки наиболее востребованы в каждой отрасли.
  * Соотношение навыков с предлагаемой зарплатой и требуемым опытом.

## Этап 5. Заключение

* Подведение итогов по каждой проверенной гипотезе (подтверждена / опровергнута / частично подтверждена).
* Общие выводы о рынке труда:

  * Географические и отраслевые особенности зарплат.
  * Зависимость зарплаты от опыта.
  * Сезонные колебания публикации вакансий.
  * Популярность цифровых навыков и тренды их спроса.
* Практические рекомендации:

  * Для соискателей: навыки, которые стоит развивать; регионы с более высокой оплатой.
  * Для образовательных учреждений: актуализация учебных программ под спрос работодателей.
  * Для HR и компаний: планирование найма, оценка конкурентоспособных зарплат.
  * Для карьерных консультантов: анализ рынка труда для помощи клиентам.
