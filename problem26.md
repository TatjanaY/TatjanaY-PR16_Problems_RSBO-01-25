# Анализ успеваемости студентов и образовательных факторов

**Ссылка на датасет:** [Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)

## Этап 1. Загрузка и предварительный обзор данных

* Загрузите CSV-файл с данными о студентах в Pandas DataFrame.
* Проверьте структуру данных:

  * `.info()` – типы и структура данных.
  * `.describe()` – базовая статистика числовых признаков.
  * `.head()` – первые строки для визуального осмотра.
* Определите ключевые признаки:

  * Демография и социальные факторы: `Gender`, `Race/Ethnicity`, `Parental_Education`, `Lunch_Type`, `Test_Prep_Course`.
  * Академические результаты: `Math_Score`, `Reading_Score`, `Writing_Score`.

## Этап 2. Очистка и предобработка данных

* Проверка пропусков и аномалий:

  * Обработка отсутствующих значений и проверка корректности диапазонов оценок (0–100).
* Создание производных признаков:

  * `overall_score` — средняя оценка по трем предметам.
  * Категории успеваемости: низкий, средний, высокий.
  * Бинарные индикаторы: выше 75-го процентиля.
* Обогащение данных:

  * Группировка уровней образования родителей в широкие категории.
  * Расчет корреляций между оценками для разных групп студентов.

## Этап 3. Разведочный анализ и проверка гипотез

| № | Гипотеза                              | Визуализация                                                                     | Тип графика |
| - | ------------------------------------- | -------------------------------------------------------------------------------- | ----------- |
| 1 | Гендерные различия в успеваемости     | Средние оценки по математике, чтению и письму для мужчин и женщин                | Barplot     |
| 2 | Влияние образования родителей         | Распределение общего балла по уровням образования родителей                      | Boxplot     |
| 4 | Эффективность подготовительных курсов | Результаты студентов с/без курсов, с разделением по уровню образования родителей | Scatterplot |
| 5 | Корреляция между предметами           | Матрица scatterplot между оценками по математике, чтению и письму                | Pairplot    |

* К каждому графику добавьте краткий вывод о подтверждении или опровержении гипотезы.
* Исследуйте взаимосвязи между демографическими и социальными факторами и результатами обучения.

## Этап 4. Создание итогового отчета

* Объедините ключевые визуализации в единый отчет с помощью `plt.subplots()`.
* Расположите графики в логической последовательности в соответствии с проверяемыми гипотезами.
* Добавьте:

  * Общее название отчета.
  * Подписи осей.
  * Поясняющие заголовки к каждому графику.
* Дополнительно:

  * Создайте интерактивную панель с `ipywidgets` для фильтрации студентов по характеристикам (пол, образование родителей, подготовительные курсы) и просмотра их результатов.

## Этап 5. Заключение

* Итоги по каждой проверенной гипотезе (подтверждена / опровергнута / частично подтверждена).
* Общие выводы:

  * Гендерные различия в успеваемости по предметам.
  * Влияние образования родителей на результаты студентов.
  * Эффективность подготовительных курсов.
  * Связь между результатами по разным предметам.
* Практические рекомендации:

  * Для преподавателей и школ: адаптация методик под разные группы студентов.
  * Для образовательных программ: учет социально-экономических факторов при проектировании курсов.
  * Для родителей и студентов: использование информации о профилях для выбора образовательной траектории.
