# Анализ киноиндустрии и кассовых сборов

**Ссылка на датасет:** [The Movies Dataset](https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset)

## Этап 1. Загрузка и предварительный обзор данных

* Загрузите CSV-файлы с информацией о фильмах, кассовых сборах, рейтингах и составе актёров в Pandas DataFrame.

* Проверьте структуру данных:

  * `.info()` – типы и структура данных.
  * `.describe()` – базовая статистика числовых признаков.
  * `.head()` – первые строки для визуального осмотра.

* Составьте краткое описание каждого признака: название, дата выхода, бюджет, сборы, жанры, актерский состав, оценки зрителей и т.д.

* Определите ключевые показатели: `Revenue` — кассовые сборы, `Budget` — бюджет, `ROI` — доходность инвестиций.

## Этап 2. Очистка, предобработка и обогащение данных

**Обработка пропусков и аномалий:**

* Проверьте пропущенные значения в критически важных полях (`Budget`, `Revenue`, `Release_Date`) и обработайте их.
* Очистите текстовые поля: жанры, ключевые слова, описание фильма — преобразуйте JSON-строки в списки.

**Создание производных признаков:**

* `Profit` = `Revenue - Budget`
* `ROI` = `(Revenue - Budget) / Budget`
* `Season` — сезон выпуска (лето, осень, зима, весна)
* `Weekday` — день недели премьеры
* `NumActors` — количество актеров
* `StarPower` — бинарный признак для известных актеров/режиссеров
* `Sentiment` — тональность сюжета/ключевых слов (анализ с TextBlob/NLTK)

**Проверка распределений:**

* Постройте гистограммы и boxplot для числовых признаков (`Revenue`, `Budget`, `ROI`) для выявления выбросов.
* Проанализируйте категориальные признаки (`Genres`, `Production_Companies`) на редкие категории.

## Этап 4. Создание итогового отчета

* Объедините ключевые визуализации из Этапа 3 в единый отчет с помощью `plt.subplots()`.

* Расположите графики в логической последовательности, соответствующей проверяемым гипотезам.

* Добавьте:

  * общее название отчета;
  * подписи осей;
  * поясняющие заголовки к каждому графику.

* По желанию: интерактивная панель с фильтрацией фильмов по жанрам, сезону, бюджету и "звездности" с помощью `ipywidgets`.

## Этап 5. Заключение

* Подведите итоги по каждой проверенной гипотезе (подтверждена / опровергнута / частично подтверждена).
* Сформулируйте общие выводы о факторах, влияющих на коммерческий успех фильмов:

  * Сезон выпуска — летние и праздничные релизы чаще успешны.
  * Бюджет — положительно влияет на сборы, но отдача снижается для сверхбюджетных фильмов.
  * "Звездность" актеров и режиссера — увеличивает сборы на старте, но не гарантирует долгосрочный успех.
  * Продолжительность фильма — умеренные фильмы чаще успешны, слишком короткие или длинные — реже.
  * ROI — различается по жанрам, некоторые нишевые жанры эффективнее массовых.
* Практические рекомендации:

  * Для студий: оптимизация бюджета, выбор сезона премьеры, подбор актеров и режиссера.
  * Для инвесторов: оценка рисков и потенциального ROI по жанрам.
  * Для креативных команд: учет аудитории при сценарии и маркетинге.
  * Для дистрибьюторов: планирование прокатной стратегии на основе данных о кассовых сборах и жанровых трендах.
