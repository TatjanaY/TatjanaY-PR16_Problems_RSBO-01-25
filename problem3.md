# Анализ рыночных данных для электронной коммерции с использованием Pandas и Matplotlib

**Ссылка на датасет:** [Superstore Sales Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

## Этап 1. Загрузка и предварительный обзор данных

* Импорт данных: загрузите `superstore_dataset.csv` в Pandas DataFrame.
* Изучение структуры данных: используйте `.info()`, `.describe()`, `.head()`.
* Краткое текстовое описание признаков и их предполагаемой роли:

  * `Order Date`, `Ship Date` – даты заказа и отгрузки, используются для анализа времени доставки и сезонности.
  * `Category`, `Sub-Category` – категориальные признаки товаров для анализа прибыльности и структуры продаж.
  * `Sales`, `Profit`, `Discount` – количественные показатели, ключевые для проверки гипотез о доходности и скидках.
  * `Region`, `Ship Mode` – географические и логистические признаки для анализа различий по регионам и способам доставки.

## Этап 2. Очистка и предобработка данных

### Обработка пропусков

* Рассчитайте процент пропусков по каждому признаку.
* Примените подходящую стратегию заполнения или удаления пропусков.

### Обработка дубликатов

* Проверьте DataFrame на полностью повторяющиеся строки и удалите их при необходимости.

### Анализ выбросов

* Выявите выбросы в числовых признаках (`Sales`, `Profit`, `Discount`) с помощью метода IQR.
* Отметьте выбросы для корректной интерпретации визуализаций.

### Создание производных признаков

* `Delivery_Days` – целое число дней между `Ship Date` и `Order Date`.
* `Order_Year_Month` – признак для агрегации по периодам (`YYYY-MM`).
* `IsProfitNegative` – бинарный признак, указывающий на убыточные заказы (`Profit < 0`).

## Этап 3. Разведочный анализ и проверка гипотез

  * Линейные графики для анализа сезонности прибыли.

  * Диаграммы Парето для кумулятивной доли прибыли по подкатегориям.
