# Анализ эффективности электронной коммерции и поведения покупателей

**Ссылка на датасет:** [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

## Этап 1. Загрузка и предварительный обзор данных

* Загрузите CSV-файлы с информацией о заказах, товарах, клиентах и отзывах в Pandas DataFrame.

* Проверьте структуру данных:

  * `.info()` – типы и структура данных.
  * `.describe()` – базовая статистика числовых признаков.
  * `.head()` – первые строки для визуального осмотра.

* Составьте краткое описание каждого признака: ID заказа, клиент, дата заказа, статус, стоимость доставки, цена товара, категория товара, отзыв, рейтинг и т.д.

* Определите ключевые показатели: `TotalOrderValue` — стоимость заказа, `FreightValue` — стоимость доставки, `ReviewScore` — оценка клиента.

## Этап 2. Очистка, предобработка и обогащение данных

**Обработка пропусков и аномалий:**

* Проверьте пропущенные значения в критических полях (`Order_Purchase_Date`, `Price`, `Freight_Value`, `Customer_State`) и обработайте их.
* Удалите дубликаты заказов.
* Приведите даты к единому формату, создайте признаки: день недели, месяц, сезон, праздничные дни.

**Создание производных признаков:**

* `TotalOrderValue` = `Price * Quantity + Freight_Value`
* `DeliveryTime` = `Order_Delivered_Date - Order_Purchase_Date`
* `ApprovalTime` = `Order_Approved_Date - Order_Purchase_Date`
* `OrderProcessingTime` = `DeliveryTime + ApprovalTime`
* `Recency` — дни с последней покупки
* `AverageOrderValue` — средний чек клиента
* `Distance` — расстояние между штатами продавца и покупателя (если возможно)

**Проверка распределений:**

* Постройте гистограммы и boxplot для числовых признаков (`TotalOrderValue`, `FreightValue`, `DeliveryTime`) для выявления выбросов.
* Проанализируйте категориальные признаки (`Customer_State`, `Product_Category`) на редкие категории.

## Этап 5. Заключение

* Подведите итоги по каждой проверенной гипотезе (подтверждена / опровергнута / частично подтверждена).

* Сформулируйте общие выводы о покупательском поведении и эффективности электронной коммерции:

  * Доставка: вес товара и расстояние важны, есть нелинейные зависимости по регионам.
  * Регион: покупательские привычки, средний чек и популярные категории различаются по штатам.
  * Оценки клиентов: отрицательно зависят от времени доставки, положительно — от скорости подтверждения заказа.
  * Средний чек и частота покупок: сильная географическая зависимость.
  * Сезонность: видны повторяющиеся пики спроса для ключевых категорий товаров.

  * Для управления ассортиментом: региональные предпочтения и сезонность в планировании.
  * Для клиентского сервиса: улучшение обслуживания и повышение оценок.
  * Для руководства: отслеживание ключевых метрик эффективности бизнеса.
