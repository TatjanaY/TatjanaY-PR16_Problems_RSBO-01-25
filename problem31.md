# Анализ энергоэффективности зданий и потребления ресурсов

**Ссылка на датасет:** [Energy Efficiency Data Set](https://www.kaggle.com/datasets/elikplim/eergy-efficiency-dataset)

## Этап 1. Загрузка и предварительный обзор данных

* Загрузите CSV-файлы с характеристиками зданий, энергопотреблением и климатическими данными в Pandas DataFrame.

* Проверьте структуру данных:

  * `.info()` – типы и структура данных.
  * `.describe()` – базовая статистика числовых признаков.
  * `.head()` – первые строки для визуального осмотра.

* Составьте краткое описание каждого признака: размеры и форма здания, площадь остекления, материалы, нагрузки на отопление/охлаждение, климатическая зона и т.д.

* Определите ключевые показатели: `Heating_Load`, `Cooling_Load`, `Total_Energy_Use`, `Compactness`, `Glazing_Ratio`.

## Этап 2. Очистка, предобработка и обогащение данных

**Обработка пропусков и аномалий:**

* Проверьте пропущенные значения и обработайте их.
* Удалите или скорректируйте выбросы в физических характеристиках и энергопотреблении.

**Создание производных признаков:**

* `Compactness` = отношение объема здания к площади поверхности
* `WindowToWallRatio` = площадь окон / площадь стен
* `RoofToFloorRatio` = площадь крыши / общая площадь
* `SpecificEnergyConsumption` = энергопотребление на единицу площади/объема
* Категориальные переменные: низкая/средняя/высокая компактность, остекление

**Обогащение данных:**

* Объедините данные о зданиях с климатическими условиями.
* Сгруппируйте здания по климатическим зонам и типам конструкций.
* Создайте сводные таблицы по энергоэффективности зданий.

## Этап 5. Заключение

* Подведите итоги по каждой проверенной гипотезе (подтверждена / опровергнута / частично подтверждена).

* Сформулируйте общие выводы о факторах, влияющих на энергоэффективность зданий:

  * Компактность: чем выше, тем меньше нагрузка на отопление.
  * Остекление: есть двойственный эффект на отопление и охлаждение.
  * Ориентация: оптимальная зависит от климатической зоны.
  * Площадь окон: после порогового значения энергопотребление резко возрастает.
  * Сочетание характеристик: ключевой фактор удельного энергопотребления.

* Практические рекомендации:

  * Для архитекторов: оптимизация формы, ориентации и площади остекления.
  * Для строителей: выбор материалов и конструкций с учетом климатической зоны.
  * Для управляющих зданиями: оценка и модернизация существующих объектов.
  * Для политиков: разработка стандартов и стимулирование энергосберегающих технологий.
  * Для исследователей: использование анализа данных для оценки энергоэффективности и выявления перспективных решений.
