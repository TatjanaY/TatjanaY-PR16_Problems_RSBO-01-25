# Анализ энергетического потребления и возобновляемых источников энергии

**Ссылка на датасет:**
👉 **World Energy Consumption** — [https://www.kaggle.com/datasets/pralabhpoudel/world-energy-consumption](https://www.kaggle.com/datasets/pralabhpoudel/world-energy-consumption) ([Baselight][1])

**Файл:** содержит таблицу `world_energy_consumption.csv` (таблица с данными по потреблению энергии, долям возобновляемых источников, выбросам, экономическим и демографическим показателям по странам и годам). ([Baselight][1])

**Описание:** Данные включают показатели энергопотребления, производства энергии разных типов, выбросов CO₂, ВВП, населения и другие энергетические и экономические метрики для большого числа стран и регионов по годам. ([Baselight][1])

## Этап 1. Загрузка и предварительный обзор данных

* Загрузите CSV‑файл `world_energy_consumption.csv` в Pandas DataFrame.
* Проверьте структуру данных:

  * `.info()` — типы и структура данных.
  * `.describe()` — базовая статистика числовых полей (`total_energy_consumption`, `renewables_consumption`, `co2_emissions`, `gdp`).
  * `.head()` — первые строки для визуального осмотра.
* Составьте краткое описание ключевых признаков:

  * `country`, `year` — страна и год;
  * `total_energy_consumption` — общее энергопотребление;
  * `renewables_consumption` — потребление энергии из ВИЭ;
  * `co2_emissions` — выбросы CO₂;
  * `gdp`, `population` — ВВП и население.
* Определите ключевые показатели для анализа: `Total_Consumption`, `Renewables_Consumption`, `CO2_Emissions`, `GDP_per_Capita` (вычисляется как `gdp / population`).

## Этап 2. Очистка, предобработка и обогащение данных

**Обработка пропусков и аномалий:**

* Проверьте пропущенные значения и обозначьте их долю по столбцам.
* При необходимости заполните или удалите пропуски (например, средними/медианой по стране или по миру).
* Убедитесь в согласованности единиц измерения для энергетических показателей.

**Создание производных признаков:**

* `Renewables_Share` = доля возобновляемых источников в общем энергопотреблении (`renewables_consumption / total_energy_consumption`).
* `Carbon_Intensity` = выбросы CO₂ на единицу энергопотребления (`co2_emissions / total_energy_consumption`).
* `Energy_Intensity` = энергопотребление на единицу ВВП (`total_energy_consumption / gdp`).
* `GDP_per_Capita` = ВВП на душу населения (`gdp / population`).
* При необходимости сгруппируйте страны по регионам или уровням дохода (например, условные группы: `High Income`, `Upper Middle`, `Lower Middle`, `Low Income`), если такая информация есть.
